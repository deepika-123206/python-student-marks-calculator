# Student Marks Calculator

A Python command-line application to automate student result processing.

### Features:
- Accepts student name and marks for 5 subjects
- Calculates Total, Average, and Grade: A/B/C/Fail
- Flags failed subjects where marks < 35
- Saves complete result to `result.txt` using File I/O

### Tech Stack & Concepts:
Python | Functions | Loops | Lists | Conditional Logic | File Handling

### How to Run:
1. Clone this repo
2. Run `python calculator.py`
3. Enter details and check output + result.txt
